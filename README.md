# dotfiles

個人PCの初期環境設定で設定ファイルを管理する

## Mac(intel)
### Requirement
- ターミナルのフォントを[SourceCodePro](./fonts)に変える

### Usage

```ShellSession
user@host: ~/dotfiles $ sh dotfilesLink.macIntel.sh
```

## Mac(apple M1)
### Requirement
- ターミナルのフォントを[SourceCodePro](./fonts)に変える

### Usage

```ShellSession
user@host: ~/workspace/dotfiles $ sh dotfilesLink.macAppleM1.sh
```
